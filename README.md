# Crypto Trading Bot V2

Long-term, evidence-driven crypto trading bot project.

## Governance

This repository is governed by **Crypto Trading Bot Development Bible v2.0 — LOCKED**.

## Core rules

- GitHub is the single source of truth.
- One bounded module/task at a time.
- No mock market data, fake success, or silent fallback.
- No module is complete without verified tests.
- Paper/demo validation is mandatory before any live-money consideration.
- Product Owner approval is required for core architecture changes and merges to `main`.
